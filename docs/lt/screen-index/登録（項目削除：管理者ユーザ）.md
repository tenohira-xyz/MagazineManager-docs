# 登録（項目削除：管理者ユーザ）

## 事前条件
- 管理者ユーザであること。

## テスト内容
1. 雑誌一覧画面の一覧1行目の「索引」リンクから遷移する。
    - 一覧に2件表示されていること。

1. 1行目の「削除」ボタンを押下する。

1. 「登録」ボタンを押す。

1. 雑誌一覧画面に遷移することを確認する。
    - URLが`/list`となっていること。
    - 管理者ユーザ用の画面レイアウトとなっていること。

1. 再度雑誌一覧画面の一覧1行目の「索引」リンクから遷移して索引画面を表示する。
    - 一覧に1件表示されていること。
    - 1行目にあった内容が表示されていないこと。
